# CampfireEffects

Get effects while near campfires.

## Config
>config-version: \
*Shows current config version, dont change.*

>check-ticks: \
*Sets how many ticks between each check for a campfire*

###### campfire:
  > effect: \
  *Sets the potion effect for the campfire*
  > level: 1\
  *
  > radius: 10

>soul-campfire:\
>  effect: DAMAGE_RESISTANCE\
>  level: 1\
>  radius: 6\
